SQL Server DML触发器

1 DML触发器的定义

DML触发器是一种通过事件触发而被自动调用并且只针对表的特殊存储过程。

2 DML触发器分类

  • after触发器(之后触发,一表多触)

  • instead of触发器(之前触发,一表一触)

after触发器要求只有执行定义操作之后才会触发，且只能定义在表上，而instead of触发器并不执行其定义的操作而是仅执行触发器本身，优先极高于触发语句的执行。

3 触发器中的虚拟表：inserted 与 deleted

  • inserted：保存即将插入或刚刚插入的数据（INSERT 和 UPDATE 时有）
  • deleted：保存被删除的数据或更新前的数据（DELETE 和 UPDATE 时有）
